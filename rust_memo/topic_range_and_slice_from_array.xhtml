Range 構造体で配列からスライスを切り出す方法について。

このメモの背景。

他の言語ではあまり見ない書き方のため、筆者はたまに忘れる…。

簡単な例

以下はどれも配列からスライスを切り出す式になっている。 


let arr = [1, 2, 3, 4, 5, 6];
let slice = &arr[..];
let slice = &arr[..3];
let slice = &arr[4..];
let slice = &arr[2..4];

補足: 上記で arr は [i32;6] の配列型、slice は &[i32] のスライス型への参照型 (動的サイズ型への参照となるのでファットポインタで管理される)。
