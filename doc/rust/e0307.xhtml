error[E0307]: invalid `self` parameter type: `Xxxx`

self の型に許可されない型を使用した。

背景情報

『組込型の特権/self への型指定』によると、self に許される型は、実装対象それ以外では、一部の組込型のみである。

サンプル

以下では、型 MyType の実装中に self の型に i32 を指定している。 


fn main() {}

struct MyType();
impl MyType {
    fn method(self: i32) {
        // nop.
    }
}
