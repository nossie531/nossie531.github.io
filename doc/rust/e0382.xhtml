ループ内のムーブ

解決策

以下の解決策はパターン A1 を例にしている。

解決策 1

参照による引数渡し

引数の型を参照型に変更する。 


fn main() {
    let var = MyType(42);
    func(&var);
    var.show_value();
}

fn func(var: &MyType) {
    _ = var;
}

struct MyType(i32);
impl MyType {
    fn show_value(self) {
        println!("val: {}", self.0)
    }
}

解決策 2

クローンによる引数渡し

引数の型に Clone を実装し、引数にクローンした値を使う。 


fn main() {
    let var = MyType(42);
    func(var.clone());
    var.show_value();
}

fn func(var: MyType) {
    _ = var;
}

#[derive(Clone)]
struct MyType(i32);
impl MyType {
    fn show_value(self) {
        println!("val: {}", self.0)
    }
}

解決策 3

コピーによる引数渡し

引数の型に Copy を実装し、引数に自動コピーされた値を使う。 


fn main() {
    let var = MyType(42);
    func(var);
    var.show_value();
}

fn func(var: MyType) {
    _ = var;
}

#[derive(Clone, Copy)]
struct MyType(i32);
impl MyType {
    fn show_value(self) {
        println!("val: {}", self.0)
    }
}
